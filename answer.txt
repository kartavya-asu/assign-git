*What is a gitignore file and what should it include for a Java project?
-The project folder can have various files. It's not necessary to keep a track record of all the file types. Files such as the ones generated by IDE, Security files, API keys, etc can be excluded in a commit by mentioning it in the gitignore file. It should include ".vscode/, .code-workspace, *.class,	*.jar,	*.war, *.ear, *.nar " for a java project.

*What does fast-forward in Git mean?
If we want to merge two branches that are on a linear path of commits without any deviation then fast-forward allows the source branch's pointer to move forward on the linear path till the destination branch's latest commit.

*Why should you have different branches?
Branches can be useful to accommodate multiple people to work on different parts of a project simultaneously.
